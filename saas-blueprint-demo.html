Embed it

Two lines. Any website.

Drop the script tag once — then place the component anywhere. Works in plain HTML, React, Vue, Webflow, WordPress, or any framework.

Basic embed 
<!-- 1. Load the component (once, in <head> or before </body>) -->
<script src="https://your-cdn.com/saas-blueprint-tracker.js"></script>

<!-- 2. Use it anywhere -->
<saas-blueprint-tracker style="height: 700px"></saas-blueprint-tracker>
With options 
<saas-blueprint-tracker
  theme="light"            <!-- "dark" (default) | "light" -->
  project="my-startup"    <!-- namespaces localStorage (run multiple trackers) -->
  style="height: 700px"   <!-- control the height -->
></saas-blueprint-tracker>
React / Next.js 
// In your component or _app.tsx
import 'saas-blueprint-tracker'  // after npm install

export default function Page() {
  return (
    <saas-blueprint-tracker
      theme="dark"
      project="acme-saas"
      style={{ height: '700px' }}
    />
  )
}
Angular 
// 1. app.module.ts — add CUSTOM_ELEMENTS_SCHEMA
import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
import 'saas-blueprint-tracker';

@NgModule({
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  // ... rest of module
})
export class AppModule {}

// 2. your.component.html
<saas-blueprint-tracker
  theme="dark"
  project="my-saas"
  style="height: 700px"
></saas-blueprint-tracker>

// 3. Listening to events — your.component.ts
import { Component, AfterViewInit, ViewChild, ElementRef } from '@angular/core';

@Component({ ... })
export class YourComponent implements AfterViewInit {
  @ViewChild('tracker') trackerRef!: ElementRef;

  ngAfterViewInit() {
    this.trackerRef.nativeElement.addEventListener('phase-complete', (e: CustomEvent) => {
      console.log('Phase complete:', e.detail); // { id, name, project }
    });
  }
}
Listening to events 
const tracker = document.querySelector('saas-blueprint-tracker')

// Fires whenever an item is checked or unchecked
tracker.addEventListener('item-toggle', e => {
  console.log(e.detail)  // { key, checked, project }
})

// Fires when all items in a phase are complete
tracker.addEventListener('phase-complete', e => {
  console.log(e.detail)  // { id, name, project }
  // e.g. send to your analytics, unlock next feature, etc.
})
What's included

Everything built in.

No backend, no accounts, no config files. Just drop it in.

💾
Auto-saved
Progress persists in localStorage. Survives refreshes, browser restarts, and page navigation.
🗂️
Multi-project
Use the project attr to run separate trackers for different SaaS ideas on the same page.
🎨
Dark & Light
Switch with theme="light". Shadow DOM keeps it isolated from your site's styles.
📡
Custom events
Hook into item-toggle and phase-complete to connect to your analytics or backend.
📱
Responsive
Sidebar collapses on small screens. Works on mobile, tablet, and desktop.
⚡
Zero deps
Pure Web Component. No React, no Vue, no build step required. One script tag.
